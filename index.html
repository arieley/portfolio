Materia escura

O projeto é uma página web criada com HTML e CSS, destinada a exibir um breve resumo sobre o tema abordado.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, imagem, conteúdo e um link para quem gostaria de saber mais.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a busca por conhecimento.

Minha Biblioteca

O projeto consiste em uma página web que simula um site voltado para a compra de livros.

O HTML estrutura o conteúdo da página, com cabeçalho, titulo, logo, um "carrossel" onde fica a imagem da capa dos livros e um rodapé. O CSS é utilizado para estilizar a página, com cores, titulos e fundo.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de livros em uma seção flexível (.livros). Cada livro é destacado em seu próprio contêiner (.livro), mostrando uma imagem da capa, e um layout flexível que se adapta a diferentes tamanhos de tela.
